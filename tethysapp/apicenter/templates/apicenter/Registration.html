Registration

Registration is the first step a user must do to access the API services. With registration the user will get a token which is then used to access the data from the system.

Following are the steps to register to the system.

  1. Go to the ICIMOD Tethys page http://tethys.icimod.org/ and click on the “Sign Up” button if not registered.
  2. Fill the form on the “Sign Up” page. It requires all the parameters like
    1. User Name (Need to login later)
    2. Email Address
    3. Password (Need to login later)
    4. CAPTCHA
  3. Once logged in, your name will be listed at the top-right portion of the page. In picture, the user is dummy user is registered as “test”. Next to the name there is “User Menu v”.
  4. On click on the user menu, a drop down page appears. Click on the “User Settings”
  5. The page will show new form to enter Full Name of the user (if not provided before).
    1. Fill the form and click on save button below.
  6. The Token will appear under the heading API Key, Please put this key in the safe place, as we will be using the key to user API service and get the data.

    Note: Don’t try to use the token from the above picture. The user “test” has been deleted from the system.
