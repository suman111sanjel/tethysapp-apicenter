Get Return Periods (2, 10, and 20-year return)

The flood recurrence interval (return period) for a stream is the probability that a flood of a given magnitude will be equaled or exceeded in a given year.

API Link http://tethys.icimod.org/apps/apicenter/hiwatAPI/ getreturnPeriodHIWATCSV/
Method GET

Parameters with description

Parameters Description Example
comid The identifier for the stream reach. 1256
cty Name of the country in small letter (nepal, bhutan or bangladesh) nepal

Example to access CSV format (Python programming)

import requests
request_params = dict(cty='nepal', comid=5)
request_headers = dict(Authorization='Token asdfqwer1234')
res = requests.get('http://tethys.icimod.org/apps/apicenter/hiwatAPI/getreturnPeriodHIWATCSV/', params=request_params, headers=request_headers)
